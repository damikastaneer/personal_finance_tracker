# personal_finance_tracker
Java



Projectconcept

FinTrack Pro — een moderne JavaFX desktop app waarmee gebruikers hun geld beheren, budgetten volgen en financiële inzichten krijgen.

Features waarmee je opvalt
MVP
Login / registratie
Dashboard met saldo, inkomsten, uitgaven
Transacties toevoegen, bewerken, verwijderen
Categorieën zoals food, rent, school, transport
Maandelijkse budgetten
Grafieken per maand/categorie
Zoek- en filterfunctie
SQLite database
Advanced features
Dark mode + light mode
CSV import/export
PDF maandrapport
Recurring transactions
Budget alerts
Savings goals
Financiële “health score”
Automatische categorisatie op basis van keywords
Multi-user support
Echt opvallend

Voeg een AI-achtige analyse toe zonder echte AI nodig te hebben:

Bijvoorbeeld:

“Je gaf deze maand 24% meer uit aan eten dan vorige maand.”

Of:

“Je spaardoel is haalbaar als je €85 per maand blijft sparen.”

Dat laat zien dat je meer doet dan alleen data opslaan.
